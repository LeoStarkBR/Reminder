# Reminder 💊

Aplicativo mobile em desenvolvimento focado em **gestão e lembretes de medicamentos**.  
O objetivo do projeto é permitir que usuários **cadastrem remédios, organizem horários e recebam lembretes para tomar suas medicações**, evitando esquecimentos e melhorando a adesão ao tratamento.

Este projeto está sendo desenvolvido durante os estudos da formação mobile da **Rocketseat**, com foco em **arquitetura de apps reais, autenticação, banco de dados e integração com serviços cloud**.

---

## ✨ Sobre o Projeto

O **Reminder** é um aplicativo que funciona como um **agenda inteligente de medicamentos**.

Com ele o usuário poderá:

- Cadastrar medicamentos
- Definir horários e frequência de uso
- Receber lembretes para tomar o remédio
- Acompanhar histórico de medicamentos cadastrados
- Gerenciar suas informações de forma segura com autenticação

A ideia é simular o desenvolvimento de um **produto mobile completo**, trabalhando com backend cloud, autenticação e persistência de dados.
